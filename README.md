> littlemcmc
>
> A lightweight and performant implementation of HMC and NUTS in Python, spun
> out of [the PyMC4 project](https://github.com/pymc-devs/pymc4). Not to be
> confused with [minimc](https://github.com/ColCarroll/minimc).
